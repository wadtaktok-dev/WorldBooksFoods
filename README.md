# 🌍 WorldBooksFoods

WorldBooksFoods is a modern Android app built with Kotlin + Jetpack Compose.

## Features
- Firebase Authentication
- Firestore Cloud Sync
- Favorite System
- Offline Cache (DataStore)
- Material 3 UI (Green Theme)
